# v3.0 Project Workspace Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the first project-workspace workflow: multi-file literal search through quickfix, quickfix navigation, quickfix panel UI, and Chapter 14 lessons.

**Architecture:** Extend the existing simulated buffers from v2.7 with quickfix metadata in `VimState`. Keep Ex parsing narrow and deterministic in `src/core/exCommands.ts`. Surface workspace status through `VimChallenge`.

**Tech Stack:** React 19, TypeScript, Vite, Vitest, i18next, Tailwind 3, existing pure Vim reducer.
