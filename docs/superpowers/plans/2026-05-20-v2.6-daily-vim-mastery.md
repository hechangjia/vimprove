# v2.6 Daily Vim Mastery Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a v2.6 foundation for daily Vim use: command-line mode, substitute workflows, Chapter 11 lessons, and a line-target minigame.

**Architecture:** Extend the pure reducer with a narrow command-line mode and delegate Ex command execution to a focused `src/core/exCommands.ts` module. Add lessons through the existing config-driven curriculum and add a new `find-target` minigame content block following the existing minigame pattern.

**Tech Stack:** React 19, TypeScript, Vite, Vitest, i18next, Tailwind 3, existing pure Vim reducer.
