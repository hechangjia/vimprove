### 2. Text Object Ninja 小游戏 [新组件]

**优先级**: 🔥 高（填补文本对象训练空白）

**游戏设计**:
```
目标: 训练 iw/aw/ip/ap/i(/i{/i[/i"/i' 等文本对象识别
玩法:
1. 屏幕显示一段代码，高亮一个目标区域
2. 玩家用最短命令选中目标（如 vi{ / dap / ciw）
3. 计时 + 评分（按命令长度 + 准确性）

难度递增:
- Level 1: 单层嵌套（word / paragraph）
- Level 2: 括号嵌套（() / {} / []）
- Level 3: 引号嵌套（"" / '' / ``）
- Level 4: 混合场景（函数调用、JSON、HTML）
```

**实现位置**:
- `src/components/minigame/TextObjectNinjaGame.tsx` — 游戏组件
- `src/data/lessons/chapter5/textobjects-ninja-game.ts` — 课程配置

**技术要点**:
- 随机生成代码片段（预设模板库）
- 高亮目标区域（CSS overlay）
- 验证用户命令是否正确选中目标
- 本地最佳成绩记录（localStorage）

**UI 设计**:
```
┌─────────────────────────────────┐
│ Text Object Ninja - Level 1     │
│ Target: Select the word "foo"   │
├─────────────────────────────────┤
│ const bar = "foo" + baz;        │
│              ^^^  ← 高亮目标     │
├─────────────────────────────────┤
│ Your command: [viw____]         │
│ Time: 3.2s  Best: 2.1s          │
│ Score: ★★★☆☆                    │
└─────────────────────────────────┘
```

### 3. VimGolf 混合模式集成 [新章节]

**优先级**: 🔥 高（真实场景 + 社区联动）

**Chapter 15: VimGolf Challenges**

**课程结构**:
```
15.1 vimgolf-intro.ts          — VimGolf 介绍与规则
15.2 vimgolf-basic-editing.ts  — 基础编辑挑战（3 题）
15.3 vimgolf-text-objects.ts   — 文本对象挑战（3 题）
15.4 vimgolf-macros.ts         — 宏挑战（2 题）
15.5 vimgolf-advanced.ts       — 进阶挑战（2 题）
```

**精选题目**（从 VimGolf 手动挑选）:
```
基础编辑:
1. "Add quotes to each line"
   - 起始: foo\nbar\nbaz
   - 目标: "foo"\n"bar"\n"baz"
   - 最优解: I"<Esc>A"<Esc>jj.

2. "Delete empty lines"
   - 起始: foo\n\nbar\n\nbaz
   - 目标: foo\nbar\nbaz
   - 最优解: :g/^$/d<Enter>

3. "Swap two words"
   - 起始: hello world
   - 目标: world hello
   - 最优解: dwwP

文本对象:
4. "Change function arguments"
   - 起始: foo(bar, baz)
   - 目标: foo(qux, quux)
   - 最优解: ci(qux, quux<Esc>

5. "Delete inside brackets"
   - 起始: [foo, bar, baz]
   - 目标: []
   - 最优解: di[

宏挑战:
6. "Format list items"
   - 起始: foo\nbar\nbaz
   - 目标: - foo\n- bar\n- baz
   - 最优解: qqI- <Esc>jq2@q

进阶:
7. "Reformat JSON"
   - 起始: {"key":"value"}
   - 目标: {\n  "key": "value"\n}
   - 最优解: （多步骤）
```

**实现位置**:
- `src/data/lessons/chapter15/` — 新建目录
- `src/data/categories.ts` — 添加 "VimGolf Challenges" 分类

**课程特性**:
- 每个挑战显示"VimGolf 官方链接"
- 显示"社区最优解"（按键数）
- 用户完成后显示"你的按键数 vs 最优解"
- 提供"查看解法提示"按钮

**i18n 翻译**:
- `src/i18n/locales/en/lessons.json` — 英文
- `src/i18n/locales/zh/lessons.json` — 中文
- `src/i18n/locales/zh-lively/lessons.json` — 中文活泼版
