# v2.7 Project Navigation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a v2.7 foundation for project navigation: simulated buffers, split/window focus commands, Chapter 12 lessons, and a `Ctrl-w` navigation minigame.

**Architecture:** Keep the existing active `state.buffer` as the visible editor surface, while adding simulated buffer/window metadata to `VimState`. Reuse `src/core/exCommands.ts` for `:ls`, buffer switching, split commands, and `:wincmd`.

**Tech Stack:** React 19, TypeScript, Vite, Vitest, i18next, Tailwind 3, existing pure Vim reducer.
