### Task 2: Add Six English Lessons

**Files:**
- Create: `src/data/lessons/chapter10/*.ts`

- [x] **Step 1: Create lesson directory**

Run:

```bash
mkdir -p src/data/lessons/chapter10
```

- [x] **Step 2: Add lessons**

Each lesson exports one `Lesson` object with:
- A stable slug matching the filename.
- `categoryId: 'chapter10'`.
- 3-5 `contentBlocks`.
- At least one `key-list`.
- A `run-example` or `challenge` where existing mini editor commands can model the workflow.

Lesson scope:
- `install-and-minimal-config`: install choices, tiny vimrc/init.lua, and practice editing config-like lines.
- `vim-in-vscode`: VSCode Vim extension habits, keymap boundaries, and settings JSON cleanup.
- `vim-in-jetbrains`: IdeaVim, IDE actions, and `.ideavimrc` concepts.
- `vim-in-terminal`: shell vi mode, commit messages, `less`, `man`, and command-line editing.
- `editor-shortcut-migration`: map common editor shortcuts to Vim equivalents.
- `realworld-refactor-demo`: combine search, operators, text objects, and repeat in one realistic cleanup.
