## 1. 插入模式快捷键设计

### 1.1 命令规格

| 命令 | 功能 | Vim 原生行为 | 实现优先级 |
|------|------|-------------|-----------|
| `Ctrl-w` | 删除前一个单词 | 删除到前一个空白或标点 | P0 |
| `Ctrl-u` | 删除到行首 | 删除当前行光标前所有字符 | P0 |
| `Ctrl-t` | 增加缩进 | 在行首插入一个 shiftwidth | P0 |
| `Ctrl-d` | 减少缩进 | 在行首删除一个 shiftwidth | P0 |

### 1.2 技术实现

**Ctrl-w 实现逻辑**:
```typescript
// src/core/utils.ts
export function deleteWordBackward(line: string, col: number): { newLine: string; newCol: number } {
  if (col === 0) return { newLine: line, newCol: 0 };
  
  let pos = col - 1;
  // 跳过当前位置的空白
  while (pos >= 0 && /\s/.test(line[pos])) pos--;
  // 删除到前一个单词边界
  while (pos >= 0 && !/\s/.test(line[pos])) pos--;
  
  const newLine = line.slice(0, pos + 1) + line.slice(col);
  return { newLine, newCol: pos + 1 };
}
```

**Ctrl-u 实现逻辑**:
```typescript
// 直接删除 line.slice(0, col)
const newLine = line.slice(col);
const newCol = 0;
```

**Ctrl-t / Ctrl-d 实现逻辑**:
```typescript
// Ctrl-t: 在行首插入 2 个空格（shiftwidth=2）
const newLine = '  ' + line;
const newCol = col + 2;

// Ctrl-d: 删除行首最多 2 个空格
let deleteCount = 0;
while (deleteCount < 2 && line[deleteCount] === ' ') deleteCount++;
const newLine = line.slice(deleteCount);
const newCol = Math.max(0, col - deleteCount);
```

### 1.3 边界情况

| 场景 | 行为 |
|------|------|
| `Ctrl-w` 在行首 | 无操作 |
| `Ctrl-u` 在行首 | 无操作 |
| `Ctrl-t` 在非行首 | 仍在行首插入缩进，光标相对位置保持 |
| `Ctrl-d` 行首无空格 | 无操作 |
| `Ctrl-w` 删除到行首 | 停在 col=0 |

### 1.4 测试用例

```typescript
describe('Insert mode shortcuts', () => {
  it('Ctrl-w deletes word backward', () => {
    let state = { ...INITIAL_STATE, buffer: ['hello world'], cursor: { line: 0, col: 11 }, mode: 'insert' };
    state = vimReducer(state, { type: 'key', key: 'w', ctrl: true });
    expect(state.buffer[0]).toBe('hello ');
    expect(state.cursor.col).toBe(6);
  });

  it('Ctrl-u deletes to line start', () => {
    let state = { ...INITIAL_STATE, buffer: ['hello world'], cursor: { line: 0, col: 5 }, mode: 'insert' };
    state = vimReducer(state, { type: 'key', key: 'u', ctrl: true });
    expect(state.buffer[0]).toBe(' world');
    expect(state.cursor.col).toBe(0);
  });

  it('Ctrl-t increases indent', () => {
    let state = { ...INITIAL_STATE, buffer: ['hello'], cursor: { line: 0, col: 2 }, mode: 'insert' };
    state = vimReducer(state, { type: 'key', key: 't', ctrl: true });
    expect(state.buffer[0]).toBe('  hello');
    expect(state.cursor.col).toBe(4);
  });

  it('Ctrl-d decreases indent', () => {
    let state = { ...INITIAL_STATE, buffer: ['  hello'], cursor: { line: 0, col: 4 }, mode: 'insert' };
    state = vimReducer(state, { type: 'key', key: 'd', ctrl: true });
    expect(state.buffer[0]).toBe('hello');
    expect(state.cursor.col).toBe(2);
  });
});
```

## 2. Text Object Ninja 游戏设计

### 2.1 游戏机制

**核心循环**:
```
1. 显示代码片段 + 高亮目标区域
2. 用户输入命令（如 viw / dap / ci"）
3. 验证命令是否正确选中目标
4. 计分：命令长度 + 时间 + 准确性
5. 进入下一关
```

**评分公式**:
```
Score = BaseScore - (KeyCount - OptimalKeyCount) * 10 - (Time - OptimalTime) * 5
BaseScore = 100
OptimalKeyCount = 最短命令长度（如 viw = 3）
OptimalTime = 3 秒
```

**星级评定**:
```
★★★★★ (5星): Score >= 90
★★★★☆ (4星): Score >= 70
★★★☆☆ (3星): Score >= 50
★★☆☆☆ (2星): Score >= 30
★☆☆☆☆ (1星): Score < 30
```
